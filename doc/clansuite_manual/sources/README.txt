This Manual is written  in ascii-doc format. To  contribute just edit the  *.asc
files  (which  are  textfiles)  from  this  directory  with  an  texteditor. The
Documentation is automatically created  on build. For further  informations view
the "clansuite_manual.asc" header.


Dieses Manual  ist im  in Ascii-Doc  Format geschrieben.  Um etwas hinzuzufügen,
einfach die *.asc Dateien (es sind Textdateien) in diesem Verzeichnis mit  einem
Texteditor bearbeiten.  Die Dokumentation  wird automatisch  beim Neubilden  von
Clansuite erstellt. Weitere Informationen sind dem Headerbereich 
der "clansuite_manual.asc" zu entnehmen.
