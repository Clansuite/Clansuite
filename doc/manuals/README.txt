   Clansuite Documentation
   -----------------------

    The Clansuite Documentation is written in ascii-doc format.
    To contribute just edit the *.asc textfiles from this directory with an texteditor.
    The Documentation is automatically generated (nightly builds).
    For further informations view the "clansuite_manual.asc" header.

    The Manuals are released under the "GNU Free Documentation License Version 1.3, 3 November 2008"
    and the "Creative Commons License: Attribution-Share Alike 3.0".


    Dieses Handbuch ist im in Ascii-Doc Format geschrieben. Um etwas hinzuzufügen,
