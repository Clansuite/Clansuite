Advanced Graphing Class Examples


This page is for demonstration purposes only, you should read the readme before trying any of the methods shown.
All these graphs were generated when this page loaded (assuming your viewing it live), most graphs are generated in under 25ms. They all have the benchmark property on to allow you to see the graphing speed. A 360 point graph takes less then 50ms.

<?php
$graph = new graph();
$graph->setProp('funcinterval',.125);
$graph->setProp('benchmark',true);
$graph->graphFunction('x^2',-10,10);
$graph->graphFunction('10sqrt(x)',-10,10,1);
$graph->graphFunction('10sin(x)',-10,10,2);
$graph->setColor('',0,'red');
$graph->setProp('title',"Function Graphing");
$graph->setColor('',2,'green');
$graph->graph();
$graph->showGraph("exa17.png");
?>
