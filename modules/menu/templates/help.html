1.1 Dynamic Tree Builder - edit

Click on the "+" or "-" near the folder icon to expand/collapse current node.
Click on the node name to select an element.
To perform an action using icons below the tree, first you have to select an element.
When you have selected an element, click Action "Info" to see it's details.

Actions:

Move Up, Move Down - move the node up or down.
Move Left - move above current node's parent.
Move Right - move inside folder that is below current node.
Insert - insert before, after or inside current node.
Info - view details, update data.
Delete - remove doc or folder (must be empty).
Update - view details: update the elements details.
Cancel - view details: cancel the details view.

1.2 Generate Menu - save

The Generation of the Menu is a two step action:
When you are done editing, you have to
(a) Generate your Menu and then
(b) Update in your database!

1.3 Restore Menu - load
