1.1 Tree Builder
Click on the "+" or "-" near the folder icon to expand/collapse current node. Click on the node name to select an element. To perform an action using icons below the tree, first you have to select an element.
Actions:

Move Up, Move Down - move the node up or down.
Move Left - move above current node's parent.
Move Right - move inside folder that is below current node.
Insert - insert before, after or inside current node.
Info - view details, update data.
Delete - remove doc or folder (must be empty).
